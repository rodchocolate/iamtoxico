## Underwear capsule
- Supplier selection: waistband customization support required (all-over or elastic print). Evaluate providers with consistent elastic quality.
- Specs: confirm printable area for waistband repeat and back placement; colorfastness and shrinkage.
- Samples: order both styles from August colors first; validate fit and print durability.
- Shopify sync: create variants by month/color; ensure hygiene return rules applied.
