# Shopify Setup

Theme
- Start with Dawn (free) for speed; evaluate paid minimal theme later.

Store config
- Domains: connect iamtoxico.com, set primary.
- Navigation: Home, Shop, About, FAQ, Contact.
- Policies: Terms, Privacy, Returns/Exchanges (Printify constraints), Shipping.
- Payments: Shopify Payments + PayPal; Apple Pay/Shop Pay enabled.
- Analytics: GA4 + Meta Pixel.

SEO
- Site title/meta, favicon, social preview.
- PDP SEO titles (Design Name — iamtoxico), alt text for images.

PDP essentials
- Size guide, fabric details (from supplier), care instructions, shipping estimates.

QA
- Test checkout, email notifications, taxes, shipping zones.
