# Agents — iamtoxico

Goal: launch a casual + loungewear micro-brand on Shopify using Printify with an initial capsule based on the included artwork.

Core constraints
- Domain ready: iamtoxico.com
- Stack: Shopify + Printify
- First asset: tshirt_sample_proportional.png (4500x5400 @ 200dpi, sRGB)

Agents and scopes

1) Brand Strategy Agent
- Define brand story, tone, vibe words (edgy, minimal, lounge-forward, unisex).
- Name lock: iamtoxico (lowercase usage), short tagline exploration.
- Color system: monochrome core + 1 accent color for each drop.
- Approve logo wordmark and favicons.
- Deliverables: brand one-pager.

2) Creative/Design Agent
- Prepare print-ready files: DTG 4500x5400 transparent PNG, 300dpi preferred; create 300dpi master if needed.
- Explore 2 placement variants (chest small, center large) and 2 colorways per garment color.
- Generate mockups via Printify and clean hero renders.
- Deliverables: print files, mockups, social cutdowns.

3) Shopify Build Agent
- Theme: select performant free theme (Dawn) or paid minimal theme.
- Configure navigation, PDP for capsule, size guide, returns, shipping, trust.
- Set up domains, favicon, SEO basics, analytics, payments, legal pages.
- Deliverables: live, password-protected store QA-ready.

4) Printify Ops Agent
- Create products: pick suppliers, colors, sizes, print areas, pricing/margins.
- Test orders, sample review, production times, shipping rules by region.
- Deliverables: products synced to Shopify, sample ordered.

5) Marketing Launch Agent
- Prelaunch signup page + socials handle check.
- Launch plan: teaser, drop date, email capture, UGC brief, influencer seeding.
- Deliverables: 2-week launch calendar, assets, email flows.

6) Legal/Policy Agent
- Draft returns/exchanges policy (Printify constraints), terms, privacy, IP.
- Trademark screening for “iamtoxico”.
- Deliverables: policy docs and checklist.

7) Analytics/Finance Agent
- Hook up GA4, Meta pixel, Shopify reports, post-purchase survey.
- Pricing model: COGS (Printify) + shipping + fees + margin targets.
- Deliverables: KPI dashboard definitions.

## Underwear capsule (Aug 2025 – Jan 2026)

- Women’s thong + Men’s boxers; waistband slogans: “I love toxico” (women), “I am toxico” (men); back print both: “handle with care”.
- 6-month schedule, 2 clashing colors per month (12 SKUs). Visual lineup complete.

Assignments
- Brand Strategy: finalize slogan styling rules (lowercase, cursive option), accent palette names.
- Creative/Design: export print-ready PNG pack per SKU; waistband vector art; back print placement.
- Printify Ops: select supplier with waistband customization; confirm print method/spec; order samples.
- Shopify Build: create collection “Underwear Capsule AW25”; product templates, size guide, hygiene return note.
- Marketing: monthly drop content calendar; captions by month; UGC brief.
- Legal: underwear-specific returns/exchanges (hygiene) policy language.
- Analytics/Finance: pricing ladder, margin model, track category performance.
